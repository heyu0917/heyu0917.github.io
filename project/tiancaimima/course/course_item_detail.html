经典国学记忆课程

Guoxue memory course

《中庸》记忆专训教材

课程介绍（共47个课程）：

“学国学的孩子不学坏，学国学的家长不失败！”国学，是中华传统文化与核心价值观，分为经部、史部、子部、集部。天才密码国学记忆课程是针对中国传统经典国学专业研发的国学记忆课程体系，中国国学经典文化底蕴丰厚，蕴藏着丰富的为人处事的经验和做人的道理。学习国学可以提升国民道德品质，培养学生良好的修养和习惯，提升学生的记忆潜能和智力等，一生受益无穷。学生通过学习国学，聆听国学，对学生的文学修养是极有帮助的，学生从小背《三字经》《孙子兵法》《道德经》《百家姓》《大学》《论语》等，不但能提升学生的记忆能力，还能提升学生的写作水平等，具有诸多优势，不仅提升自己的人文修养，还可掌握一套强大的记忆方法、学习方法，对学生的健康成长是非常有利的。

《经典国学课程》共有47个单本课程，是《天才密码记忆课程》的应用延伸，主要针对部分学员学完《天才密码记忆课程》无法应用到其它书本背诵而独立研发的课程。研发这套课程的主要目的是解决广大学员不能将《天才密码记忆课程》运用到课本和其它学科背诵而设立的具有针对性、实用性强的课程。学习《经典国学课程》须掌握《天才密码记忆课程》作为基础，才能将所选对应的国学书本倒背如流。

学习国学经典的好处：

1、快乐成长身心健康，培养良好思想品德；
2、提升学生智力、记忆、语言以及学习力；
3、善于动脑，提升思考力、悟性创造力等；
4、长得开口说得出话，拿得起笔写得出文；
5、动静分明，全面提升各科目的学习成绩；
6、有主见，培养学生感恩心，时间观念等；
7、人格优化行为和谐，与人为善懂礼敬人；
8、培养行为习惯，促进家庭成长，社会进步；
9、学以致用，提升升学率为幸福奠定基础。

“天才密码”国学体系优势：

1、运用先进图像记忆法，用时少效果好；
2、课程生动有趣，记得牢，终生不会忘；
3、结合记忆宫殿，正背倒背抽背样样顺；
4、授予强大的思维分析方法，助力成长；
5、合理时间安排，因材施教，学以致用。

课程科目：

1、经部——儒学经典：《周一》、《尚书》、《周礼》、《礼记》、《仪礼》、《诗经》、《春秋左传》、《春秋公羊传》、《春秋谷梁传》、《论语》、《孝经》、《尔雅》、《孟子》。
2、史部——各种体裁历史著作：《史记》、《汉书》、《后汉书》、《三国志》、《资治通鉴》、《战国策》、《宋元明史纪事本末》。
3、子部——诸子百家及释道宗教著作：《老子》、《墨子》、《庄子》、《荀子》、《韩非子》、《管子》、《尹文子》、《慎子》、《公孙龙子》、《淮南子》、《抱朴子》、《列子》、《孙子》、《山海经》、《艺文类聚》、《金刚经》、《四十二章经》。
4、集部——诸子百家及艺术、谱录：《楚辞》、《全唐诗》，《全宋词》，《乐府诗集》、《文选》、《李太白集》、《杜工部集》、《韩昌黎集》、《柳河东集》、《白香山集》。

学习效果：

1、将所选对应的国学书本倒背如流。
2、 任意抽背，能够说出第几章第几页。

学习时间：

1、《道德经》42课时。
2、《孙子兵法》42课时。
3、《大学》30课时。

学员要求：

8岁以上青少年和成年人。
